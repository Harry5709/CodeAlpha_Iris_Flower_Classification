# Iris Flower Classification 🌸

A Machine Learning project built during my **CodeAlpha Data Science Internship** to classify Iris flower species based on their physical measurements.

## 📋 Project Breakdown
* **A. Environment Setup:** Configuring Scikit-Learn, Pandas, and Seaborn.
* **B. Load the Data:** Ingesting the `Iris.csv` external dataset.
* **C. Data Cleaning:** Removing non-essential features like the tracking `Id`.
* **D. Exploratory Data Analysis (EDA):** Plotting sepal dimensions and calculating correlation heatmaps.
* **E. Train-Test Split:** Allocating an 80/20 data layout for model validation.
* **F. Model Training:** Applying the K-Nearest Neighbors (KNN) classification algorithm.
* **G. Evaluation:** Achieving high accuracy checked via standard metrics.

## 🛠️ Technologies Used
* Python 3
* Jupyter Notebook
* Pandas & NumPy
* Matplotlib & Seaborn
* Scikit-Learn
